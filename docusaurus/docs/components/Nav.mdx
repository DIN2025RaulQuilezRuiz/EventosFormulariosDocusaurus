# Nav

Responsive mobile navigation component.

## Description

Renders a burger button that toggles a mobile navigation menu and a set of NavLink items.
- Manages an "open" boolean state to toggle the menu.
- Accessible: Updates `aria-expanded`, `aria-controls`, and `aria-label`.
- Responsive: Toggles `nav-mobile-open` class on the nav element.

## Usage

```jsx
import Navigate from '../src/components/Nav';

<Navigate />
```

## Routes

Links to:
- `/inicio`
- `/tienda`
- `/admin`

## Props

This component does not accept any props.
